## Equal-data 模块说明

### 概述

`equal-data` 是一个财经数据接口包，主要用于查询 A 股股票及相关主体的专业金融数据（如每日成交价格、高管持股变动、机构动向、龙虎榜、策略表现等）。
该模块通过标准化 API 方式统一了数据资产的对外服务方式，帮助有需要的技术用户以更**实时、简洁、轻量**的方式使用相关数据。

### 快速上手

- 安装 Python 运行环境（推荐 Python 3.7+），并安装 `equal-data` 依赖包（推荐从清华 PyPI 镜像安装）：

```bash
pip install equal-data -i https://pypi.tuna.tsinghua.edu.cn/simple
```

- 根据接口文档，使用 Python 代码获取数据（例如：**高管增减持金额排行榜** 接口）：

```python
from kjj import KjjApi

api = KjjApi('bear your token')

# 高管持股变动-增减持金额排行榜列表
data = api.query_kjj_data(
    interfaceId="14",
    period=None,        # Integer  # 默认: 1
    changeType=None,    # Integer  # 默认:
    pageIndex=None,     # Integer  # 默认: 0
    pageSize=None,      # Integer  # 默认: 10
    selfSelected=None,  # boolean  # 默认: false
    sortDesc=None,      # String   # 默认: 1-DESC
)
```

### 通用参数格式说明

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId`  | `String`  | 是 | 如 `14` | 接口 ID，对应下方索引表中某一行的接口编号 |
| 其他业务参数   | 见各接口文档 | - | 见各接口默认值 | 具体含义与默认值以单个接口的 `请求参数` 表为准 |

- **返回格式**：统一为 JSON，对应各接口 `返回参数` 表结构。

### 功能与使用场景

- **核心功能**：通过 `equal-data` PyPI 包，调用看究竟的标准化金融数据 API，获取历史及实时数据。
- **典型使用方式**：当用户需要获取 A 股股票、高管持股、机构动向、龙虎榜、交易策略等历史或实时数据时，由智能体选择合适的接口（本索引中的某一行），并通过 `query_kjj_data` 发起调用。
