# 文章信息

> 分类: 资讯信息 | 目录: `资讯信息` | 索引见 `SKILL.md`


### 描述

文章信息

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "NEWS_INFO_ARTICLE_LIST_D6DA008D" | 接口ID |
| `page` | `Integer` | 否 | - | 页码 示例：0 |
| `pageSize` | `Integer` | 否 | - | 每页条数 示例：10 |
| `keyword` | `String` | 否 | - | 关键词 示例：泡泡玛特 |
| `startTime` | `String` | 否 | - | 开始时间 示例：2020-01-01 |
| `endTime` | `String` | 否 | - | 结束时间 示例：2024-12-31 |

### 返回参数

文章信息列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `publishTime` | `Date` | 否 | 发布时间 |
| `title` | `String` | 否 | 标题 |
| `url` | `String` | 否 | 文章链接 |
| `description` | `String` | 否 | 导语 |

```python
from equal_data import KjjApi

    keyword=None,  # String  # 默认: 
    startTime=None,  # String  # 默认: 
    endTime=None,  # String  # 默认: 
)
```
