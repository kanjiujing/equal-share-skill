# 快讯信息（近一个月的快讯）

> 分类: 资讯信息 | 目录: `资讯信息` | 索引见 `SKILL.md`


### 描述

快讯信息（近一个月的快讯）

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "47" | 接口ID |
| `page` | `Integer` | 否 | - | 页码 |
| `pageSize` | `Integer` | 否 | - | 每页条数 |
| `keyword` | `String` | 否 | - | 关键词 |
| `startTime` | `String` | 否 | - | 开始时间 |
| `endTime` | `String` | 否 | - | 结束时间 |

### 返回参数

快讯信息列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `publishTime` | `Date` | 否 | 发布时间 |
| `title` | `String` | 否 | 标题 |
| `content` | `String` | 否 | 内容 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
token = os.getenv('KJJ_API_KEY')  
# 调用接口
data = api.query_kjj_data(
    interfaceId="47",
    page=None,  # Integer  # 默认: 
    pageSize=None,  # Integer  # 默认: 
    keyword=None,  # String  # 默认: 
    startTime=None,  # String  # 默认: 
    endTime=None,  # String  # 默认: 
)
```
