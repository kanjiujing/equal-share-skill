# 指数基本信息列表-根据交易倒序排序

> 分类: 指数相关数据 | 目录: `指数相关数据` | 索引见 `SKILL.md`


### 描述

获取指数基本信息列表，按最近交易日倒序排序

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "I1" | 接口ID |
| `indexCodes` | `str` | 否 | - | 指数代码,多个用逗号隔开 示例:'000001','000002' |
| `indexNames` | `str` | 否 | - | 指数名称,多个用逗号隔开-精确匹配, 示例:'上证指数','深证成指' |
| `pageIndex` | `int` | 是 | `0` | 当前页从0开始:不填系统默认0 |
| `pageSize` | `int` | 是 | `10` | 每页显示记录数,最大不能超过200,不填系统默认10 |

### 返回参数

ResultData.data 为 List，指数基本信息列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `indexCode` | `str` | 否 | 指数代码 |
| `indexFullName` | `str` | 否 | 指数全称 |

### 调用示例

```python
from equal_data import EqualDataApi  
api = EqualDataApi('your API_KEY') 
# 调用接口
data = api.query_equal_data(
    interfaceId="I1",
    indexCodes=None,  # str  # 默认: 
    indexNames=None,  # str  # 默认: 
    pageIndex=None,  # int  # 默认: 0
    pageSize=None,  # int  # 默认: 10
)
```
