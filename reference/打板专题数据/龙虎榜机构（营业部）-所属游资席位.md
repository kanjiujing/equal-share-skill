# 龙虎榜机构（营业部）-所属游资席位

> 分类: 打板专题数据 | 目录: `打板专题数据` | 索引见 `SKILL.md`


### 描述

根据营业部全称查询所属游资席位列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "36" | 接口ID |
| `deptName` | `String` | 是 | - | 营业部全称 |

### 返回参数

龙虎榜机构（营业部）对应的游资席位名称列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `hotMoneyNames` | `List<String>` | 是 | 游资席位名称列表 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
token = os.getenv('KJJ_API_KEY')  
# 调用接口
data = api.query_kjj_data(
    interfaceId="36",
    deptName=None,  # String  # 默认: 
)
```
