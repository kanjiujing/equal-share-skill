# 机构动向全量数据

> 分类: 机构动向 | 目录: `机构动向` | 索引见 `SKILL.md`


### 描述

提供 公募基金, 私募基金, 外资,国家队,保险公司, 信托公司, 证券公司以及知名投资机构进入上市公司前十大股东和前十大流通股东的最新资金动向

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "J1" | 接口ID |
| `pageIndex` | `int` | 否 | `0` | 当前页从0开始，默认0 |
| `pageSize` | `int` | 否 | `10` | 每页显示记录数，默认10 |
| `agencyType` | `int` | 否 | - | 机构类型 3001-公募 1003-私募 1006-信托公司 1007-证券公司 ，1009-保险公司 1015-外资 8000-国家队 1201-知名机构，不传不限 |
| `changeDirect` | `int` | 否 | - | 持股变动类型 1-新进 2-退出 3-增持 4-减持，不传为全部 |
