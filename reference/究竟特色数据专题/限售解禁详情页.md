# 限售解禁详情页

> 分类: 究竟特色数据专题 | 目录: `究竟特色数据专题` | 索引见 `SKILL.md`


### 描述

限售解禁详情页

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "DRAGON_TIGER_FREE_BATCH_STOCKCODE_DETAIL_1CA5561F" | 接口ID |
| `stockCode` | `String` | 是 | - | 股票代码 |

### 返回参数

限售解禁详情页

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `scaleType` | `Integer` | 否 | 公司标签  市值、管理规模类型 1-千亿公募 2-百亿私募 3-500亿市值上市公司 |
| `totalShares` | `Long` | 否 | 公司总股本 |
| `circulateShares` | `Long` | 否 | 流通股本 |
| `planTotalQuantity` | `Long` | 否 | 预告解禁总数量 |
| `historyTotalQuantity` | `Long` | 否 | 历史解禁总数量 |
| `historyNum` | `Integer` | 否 | 近10年历史发生次数 |
| `detailList` | `List<FreeBatchDetailDataVo>` | 否 | 解禁明细 |

**`detailList`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `freeDate` | `String` | 否 | 解禁日期 |
| `managerNum` | `Long` | 否 | 涉及股东数量 |
| `totalQuantity` | `Long` | 否 | 本次解禁总股数 |
| `dataType` | `Integer` | 否 | 1-解禁预告 ; 2-历史解禁 |
| `ratioOfTotal` | `BigDecimal` | 否 | 解禁数量占总股本比 |
| `totalMarketValue` | `BigDecimal` | 否 | 预计解禁市值 |
| `label` | `String` | 否 | 是否今日解禁 |
| `totalShares` | `Long` | 否 | 总股本 |
| `managerInfoList` | `List<FreeBatchManagerInfoVo>` | 否 | 涉及股东明细 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
# 调用接口
data = api.query_kjj_data(
    interfaceId="DRAGON_TIGER_FREE_BATCH_STOCKCODE_DETAIL_1CA5561F",
    stockCode=None,  # String  # 默认: 
)
```
