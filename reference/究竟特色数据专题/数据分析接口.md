# 数据分析接口

> 分类: 究竟特色数据专题 | 目录: `究竟特色数据专题` | 索引见 `SKILL.md`


### 描述

数据分析接口

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "26" | 接口ID |
| `stockCode` | `String` | 是 | - | 股票代码 |

### 返回参数

数据分析接口

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `oddsStrackDayNum` | `Integer` | 否 | 上涨概率跟踪天数 |
| `zdfAvgStrackDayNum` | `Integer` | 否 | 平均涨幅跟踪天数 |
| `zdfOdds` | `BigDecimal` | 否 | 上涨概率 |
| `zdfAvg` | `BigDecimal` | 否 | 平均涨幅 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="26",
    stockCode=None,  # String  # 默认: 
)
```
