# 机构动向-机构详情

> 分类: 究竟特色数据专题 | 目录: `究竟特色数据专题` | 索引见 `SKILL.md`


### 描述

机构动向-机构详情

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "24" | 接口ID |
| `agencyOnlyId` | `String` | 是 | - | agencyOnlyId |
| `agencyName` | `String` | 是 | - | 机构名称 |
| `agencyType` | `Integer` | 是 | - | 机构类型 |

### 返回参数

机构动向-机构详情 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `agencyName` | `String` | 是 | 机构名称 |
| `fundCompanyName` | `String` | 否 | 基金公司名称 |
| `headImage` | `String` | 否 | 机构头像地址 |
| `synopsis` | `String` | 否 | 机构简介 |
| `agencyTypesArr` | `JSONArray` | 否 | 机构类型数组 |
| `fundCompanyTypesArr` | `JSONArray` | 否 | 基金公司类型数组 |
| `infoId` | `String` | 否 | 机构infoId |
| `infoType` | `Integer` | 否 | 机构信息类型 |
| `agencyType` | `Integer` | 否 | 机构类型代码 |
