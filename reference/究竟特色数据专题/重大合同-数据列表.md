# 重大合同-数据列表

> 分类: 究竟特色数据专题 | 目录: `究竟特色数据专题` | 索引见 `SKILL.md`


### 描述

重大合同-数据列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "32" | 接口ID |
| `pageIndex` | `Integer` | 是 | - | 当前页 从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页容量 |

### 返回参数

重大合同-数据列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `publicTime` | `String` | 否 | 发布时间 |
| `odds` | `BigDecimal` | 否 | 上涨下跌概率 |
| `contractAmount` | `BigDecimal` | 否 | 合同金额 |
| `trackDayNum` | `Integer` | 否 | 跟踪天数 |
| `signBody` | `String` | 否 | 签署主体 |
| `signBodyRelative` | `String` | 否 | 签署主体与上市公司关系 |
| `otherSigner` | `String` | 否 | 其他签署方 |
| `contractNum` | `Integer` | 否 | 历史合同数 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
token = os.getenv('KJJ_API_KEY')  
# 调用接口
data = api.query_kjj_data(
    interfaceId="32",
    pageIndex=None,  # Integer  # 默认: 
    pageSize=None,  # Integer  # 默认: 
)
```
