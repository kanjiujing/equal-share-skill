# 高管持股-详情页基本信息（含数据分析）

> 分类: 究竟特色数据专题 | 目录: `究竟特色数据专题` | 索引见 `SKILL.md`


### 描述

获取高管持股-详情页基本信息（含数据分析）

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "19" | 接口ID |
| `stockCode` | `String` | 是 | - | 股票代码 |

### 返回参数

高管持股-详情页基本信息（含数据分析）

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `companyName` | `String` | 否 | 公司全称 |
| `totalQuantity` | `Long` | 否 | 增减持数量 |
| `totalAmount` | `BigDecimal` | 否 | 总金额 |
| `ratioOfTotal` | `BigDecimal` | 否 | 占总股本比 |
| `ratioOfCirculate` | `BigDecimal` | 否 | 占流通股比 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
token = os.getenv('KJJ_API_KEY')  
# 调用接口
data = api.query_kjj_data(
    interfaceId="19",
    stockCode=None,  # String  # 默认: 
)
```
