# 业绩预告-业绩暴雷、业绩预增

> 分类: 究竟特色数据专题 | 目录: `究竟特色数据专题` | 索引见 `SKILL.md`


### 描述

业绩预告-业绩暴雷、业绩预增

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "31" | 接口ID |
| `preType` | `Integer` | 否 | - | 预告类型 0-预减 1-预增(默认为0) |
| `currentPage` | `Integer` | 是 | - | 当前页 从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页容量 |

### 返回参数

业绩预告-业绩暴雷、业绩预增

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `logo` | `String` | 否 | 股票logo |
| `label` | `List<String>` | 否 | 标签列表 |
| `predictAmtMin` | `Long` | 否 | 预测数值最小 |
| `predictAmtMax` | `Long` | 否 | 预测数值最大 |
| `addAmpMin` | `BigDecimal` | 否 | 业绩变动同比最小 |
| `addAmpMax` | `BigDecimal` | 否 | 业绩变动同比最大 |
| `predictType` | `String` | 否 | 预告类型 |
| `publicDate` | `String` | 否 | 预告日期 |
| `lossNum` | `Integer` | 否 | 亏损事件数量 |
| `trackDayNum` | `Integer` | 否 | 跟踪天数 |
| `odds` | `BigDecimal` | 否 | 上涨下跌概率 |
| `zdfAvg` | `BigDecimal` | 否 | 平均涨跌幅 |
| `zdfAvgMax` | `BigDecimal` | 否 | 平均最高涨跌幅 |
| `desc` | `String` | 否 | 描述 |
