# 解禁数据列表（历史解禁/预告/仅自选/详情页）

> 分类: 究竟特色数据专题 | 目录: `究竟特色数据专题` | 索引见 `SKILL.md`


### 描述

解禁数据列表（历史解禁/预告/仅自选/详情页）

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "24" | 接口ID |
| `pageIndex` | `Integer` | 是 | - | 当前页从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页显示记录数 |
| `selfSelected` | `boolean` | 否 | `false` | 仅自选 true: 只看自选 false：全部（默认值） |
| `freeDate` | `String` | 否 | - | 日期筛选，格式： 2025-09 |
| `sortDesc` | `String` | 否 | `1-DESC` | 排序描述 ，请给 1-ASC这种格式 ，ASC：升序  DESC： 降序 其中1表示 解禁日期 <br>2表示 占总股本比例 |
| `dataType` | `Integer` | 否 | `1` | 1-解禁预告 2-历史解禁 |

### 返回参数

解禁数据列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `esId` | `Integer` | 否 | 主键ID |
| `infoId` | `String` | 否 | 客户端要传的值 |
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `odds` | `BigDecimal` | 否 | 上涨概率 |
| `trackDayNum` | `Integer` | 否 | 跟踪天数 |
| `historyNum` | `Integer` | 否 | 历史发生次数 |
| `freeDate` | `String` | 否 | 解禁日期 |
| `scaleType` | `Integer` | 否 | 公司标签 ， 3-表示 500以上市值 |
| `totalQuantity` | `Long` | 否 | 解禁数量 |
| `totalShares` | `Long` | 否 | 公司总股本 |
| `circulateShares` | `Long` | 否 | 流通股本 |
| `totalMarketValue` | `BigDecimal` | 否 | 预计解禁市值 |
| `actualMarketValue` | `BigDecimal` | 否 | 实际解禁市值 |
| `ratioOfTotal` | `BigDecimal` | 否 | 解禁数量占总股本比 |
| `sharesType` | `String` | 否 | 解禁股份来源 |
| `label` | `String` | 否 | 是否今日解禁 |
| `managerName` | `String` | 否 | 股东名称 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="24",
    pageIndex=None,  # Integer  # 默认: 
    pageSize=None,  # Integer  # 默认: 
    selfSelected=None,  # boolean  # 默认: false
    freeDate=None,  # String  # 默认: 
    sortDesc=None,  # String  # 默认: 1-DESC
    dataType=None,  # Integer  # 默认: 1
)
```
