### 描述

分红送转-预案/实施

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "33" | 接口ID |
| `reportTime` | `String` | 是 | - | 报告期 |
| `scheduleType` | `Integer` | 是 | - | 实施进度 1-分红预案 2-实施完成 |
| `currentPage` | `Integer` | 是 | - | 当前页 从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页容量 |

### 返回参数

分红送转-预案/实施

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `odds` | `BigDecimal` | 否 | 上涨下跌概率 |
| `trackDayNum` | `Integer` | 否 | 跟踪天数 |
| `plan` | `String` | 否 | 分配方案 |
| `schedule` | `String` | 否 | 实施进度 |
| `depriveRightsDate` | `String` | 否 | 除权除息日 |
| `equityRegistDate` | `String` | 否 | 股权登记日 |
| `reportDate` | `String` | 否 | 报告期 |
| `publicDate` | `String` | 否 | 预案公告日期 |
| `implPublicDate` | `String` | 否 | 实施公告日期 |
