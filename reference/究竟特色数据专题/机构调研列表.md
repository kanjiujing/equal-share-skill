# 机构调研列表

> 分类: 究竟特色数据专题 | 目录: `究竟特色数据专题` | 索引见 `SKILL.md`


### 描述

机构调研列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "34" | 接口ID |
| `stockCode` | `String` | 是 | - | 股票代码 |
| `dateStr` | `String` | 是 | - | 公告日期 |
| `pageIndex` | `Integer` | 是 | - | 当前页 从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页容量 |

### 返回参数

机构调研列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `researchDate` | `String` | 否 | 调研日期 |
| `reportDate` | `String` | 否 | 公告日期 |
| `researchFirstTradeDate` | `String` | 否 | 调研后第一个交易日 |
| `starNum` | `Integer` | 否 | 星标数量 |
