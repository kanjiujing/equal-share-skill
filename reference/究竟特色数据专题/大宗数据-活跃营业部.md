# 大宗数据-活跃营业部

> 分类: 究竟特色数据专题 | 目录: `究竟特色数据专题` | 索引见 `SKILL.md`


### 描述

大宗数据-活跃营业部

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "29" | 接口ID |
| `dataType` | `Integer` | 否 | `1` | 数据类型 1-大宗溢价 2-大宗折价 |
| `pageIndex` | `Integer` | 否 | `0` | 当前页 从0开始 |
| `pageSize` | `Integer` | 否 | `10` | 每页容量 |

### 返回参数

大宗数据-活跃营业部

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `buyerDeptName` | `String` | 否 | 营业部名称 |
| `dateStr` | `String` | 否 | 交易日期 |
| `odds` | `BigDecimal` | 否 | 上涨或下跌概率 |
| `trackDayNum` | `Integer` | 否 | 跟踪天数 |
| `num` | `Integer` | 否 | 近10年折价或溢价次数 |
| `zdfAvg` | `BigDecimal` | 否 | 平均涨幅 |
| `dealAmount` | `Long` | 否 | 平均涨幅 |
| `stockCodes` | `String` | 否 | 股票代码列表 |
| `stockList` | `List<Stock>` | 否 | 操作过的股票列表（股票代码、股票名称） |

**`stockList`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |

from equal_data import KjjApi
