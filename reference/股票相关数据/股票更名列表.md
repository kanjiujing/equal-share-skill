# 股票更名列表

> 分类: 股票基础数据 | 目录: `股票相关数据` | 索引见 `SKILL.md`


### 描述

获取股票更名列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "3" | 接口ID |
| `stockCode` | `List<String>` | 否 | - | 股票代码/名称 |
| `pageNum` | `Integer` | 否 | `0` | 分页偏移量，从 0 开始（第 1 页为 0，第 2 页为 pageSize） |
| `pageSize` | `Integer` | 否 | `10` | 每页条数 |

### 返回参数

ResultData.data 为 List，元素为股票更名

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `companyName` | `String` | 否 | 公司名称 |
| `formername` | `String` | 否 | 曾用名 |
| `foundDate` | `Date` | 否 | 成立日期 |
| `listingDate` | `Date` | 否 | 上市日期 |
| `changeReason` | `String` | 否 | 更名原因 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="3",
    stockCode=None,  # List<String>  # 默认: 
    pageNum=None,  # Integer  # 默认: 0
    pageSize=None,  # Integer  # 默认: 10
)
```
