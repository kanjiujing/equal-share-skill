# 热门概念板块-股票列表

> 分类: 股票基础数据 | 目录: `股票相关数据` | 索引见 `SKILL.md`


### 描述

获取热门概念板块-股票列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "6" | 接口ID |
| `tradeDate` | `String` | 否 | - | 交易日期 |
| `themeLevel` | `Integer` | 否 | - | 概念板块级别（1：一级板块, 2：二级板块, 3：三级板块, 默认） |
| `pageNum` | `Integer` | 否 | `0` | 分页偏移量，从 0 开始（第 1 页为 0，第 2 页为 pageSize） |
| `pageSize` | `Integer` | 否 | `10` | 每页条数 |

### 返回参数

ResultData.data 为 List，元素为热门概念板块-股票列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `nameLevelOne` | `String` | 否 | 板块/概念名称 |
| `nameLevelTwo` | `String` | 否 | 所属板块/概念二级分类 |
| `nameLevelThree` | `String` | 否 | 所属板块/概念三级分类 |
| `industryChgPct` | `String` | 否 | 板块/概念涨跌幅(%) |
| `stockName` | `String` | 否 | 股票名称 |
| `stockCode` | `String` | 否 | 股票代码 |
| `latestChgPct` | `String` | 否 | 最新涨跌幅(%) |
| `themeIntroduction` | `String` | 否 | 概念板块简介 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="6",
    tradeDate=None,  # String  # 默认: 
    themeLevel=None,  # Integer  # 默认: 
    pageNum=None,  # Integer  # 默认: 0
    pageSize=None,  # Integer  # 默认: 10
)
```
