# 每日基本面指标数据

> 分类: 股票行情数据 | 目录: `股票行情数据` | 索引见 `SKILL.md`


### 描述

每日基本面指标数据

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "43" | 接口ID |
| `stockCodeList` | `List<String>` | 否 | - | 股票代码列表 |
| `tradeDate` | `String` | 否 | - | 交易日期 |
| `startDate` | `String` | 否 | - | 开始日期 |
| `endDate` | `String` | 否 | - | 结束日期 |

### 返回参数

ResultData.data 为 List，元素为每日基本面指标数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `tradeDate` | `String` | 否 | 交易日期 |
| `closePrice` | `Bigdecimal` | 否 | 收盘价 |
| `turnoverRate` | `Bigdecimal` | 否 | 换手率 |
| `quantityRelative` | `Bigdecimal` | 否 | 量比 |
| `peTTM` | `Bigdecimal` | 否 | 市盈率（TTM） |
| `pb` | `Bigdecimal` | 否 | 市净率 |
| `psTTM` | `Bigdecimal` | 否 | 市销率TTM |
| `dividendYieldTTM` | `Bigdecimal` | 否 | 股息率%（TTM） |
| `totalShares` | `Bigdecimal` | 否 | 总股本（万股） |
| `circulateShares` | `Bigdecimal` | 否 | 流通股本（万股） |
| `totalMarketValue` | `Bigdecimal` | 否 | 总市值（万元） |
| `circulateMarketValue` | `Bigdecimal` | 否 | 流通市值（万元） |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
# 调用接口
data = api.query_kjj_data(
    interfaceId="43",
    stockCodeList=None,  # List<String>  # 默认: 
    tradeDate=None,  # String  # 默认: 
    startDate=None,  # String  # 默认: 
    endDate=None,  # String  # 默认: 
)
```
