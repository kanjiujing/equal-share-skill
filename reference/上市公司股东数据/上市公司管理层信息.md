# 上市公司管理层信息

> 分类: 上市公司股东数据 | 目录: `上市公司股东数据` | 索引见 `SKILL.md`


### 描述

上市公司管理层信息：按股票代码/名称（可多选）、排名日期、排名等条件动态筛选，整体分页返回上市公司管理层信息。

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "44" | 接口ID |
| `stockCodeList` | `List<String>` | 是 | - | 股票代码列表,支持单个和多个 |
| `managerName` | `String` | 否 | - | 高管姓名 |
| `position` | `String` | 是 | - | 职位（董事长、董秘、总经理） |

### 返回参数

ResultData.data 为 List，元素为 ListedCompanyManagerInfoVo

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `managerName` | `String` | 否 | 高管姓名 |
| `companyName` | `String` | 否 | 公司名称 |
| `position` | `String` | 否 | 职位（董事长、董秘、总经理） |
| `sex` | `String` | 否 | 性别 |
| `age` | `Integer` | 否 | 年龄 |
| `education` | `String` | 否 | 学历 |
| `introduction` | `String` | 否 | 简介 |
| `salary` | `String` | 否 | 薪资 |
| `workExperienceList` | `List<ManagerExperienceVo>` | 否 | 工作经历列表 |

**`workExperienceList`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `startDate` | `String` | 否 | 任职开始时间 |
| `endDate` | `String` | 否 | 任职结束时间 |
| `companyName` | `String` | 否 | 公司名称 |
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `position` | `String` | 否 | 职位 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
token = os.getenv('KJJ_API_KEY')  
# 调用接口
data = api.query_kjj_data(
    interfaceId="44",
    stockCodeList=None,  # List<String>  # 默认: 
    managerName=None,  # String  # 默认: 
    position=None,  # String  # 默认: 
)
```
