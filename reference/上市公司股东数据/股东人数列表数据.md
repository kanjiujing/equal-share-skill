# 股东人数列表数据

> 分类: 上市公司股东数据 | 目录: `上市公司股东数据` | 索引见 `SKILL.md`


### 描述

股东人数列表数据：股票代码动态筛选，返回上市公司股东人数列表数据。

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "LISTEDCOMPANY_HOLDER_NUM_6ED90164" | 接口ID |
| `stockCode` | `String` | 是 | - | 股票代码 |
| `dataDate` | `String` | 否 | - | 数据日期 |

### 返回参数

ResultData.data 为 List，元素为 HolderNumListVo

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `dataDate` | `String` | 否 | 数据日期 |
| `holderTotalNum` | `String` | 否 | 股东总户数 |
| `totalNumRatio` | `String` | 否 | 股东总户数增减比例(%) |
| `avgFreeShares` | `String` | 否 | 户均持股数（自由流通股） |
| `avgFreesharesRatio` | `String` | 否 | 户均持股数增减比例(%) |
| `avgHoldAmt` | `String` | 否 | 户均持股市值 |
| `holdRatioTotal` | `String` | 否 | 持股比例合计(%) |
| `freeholdRatioTotal` | `String` | 否 | 自由流通股持股比例合计(%) |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
# 调用接口
data = api.query_kjj_data(
    interfaceId="LISTEDCOMPANY_HOLDER_NUM_6ED90164",
    stockCode=None,  # String  # 默认: 
    dataDate=None,  # String  # 默认: 
)
```
