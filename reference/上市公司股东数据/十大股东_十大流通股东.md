# 十大股东/十大流通股东

> 分类: 上市公司股东数据 | 目录: `上市公司股东数据` | 索引见 `SKILL.md`


### 描述

十大股东/十大流通股东：按股票代码/名称（可多选）、数据日期、股东类型等条件动态筛选，整体分页返回上市公司十大股东、十大流通股东数据。

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "45" | 接口ID |
| `stockCode` | `String` | 否 | - | 股票代码 |
| `dataDate` | `String` | 是 | - | 数据日期 |
| `holderType` | `Integer` | 是 | - | 股东类型 0-十大股东 1-十大流通股东 |

### 返回参数

ResultData.data 为 List，元素为 TenShareholderVo

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `shareholderName` | `String` | 否 | 股东名称 |
| `shareholdNum` | `String` | 否 | 持股数 |
| `shareholdRatio` | `String` | 否 | 占总股本比 |
| `changeRatio` | `String` | 否 | 变动比例 |
| `shareholdNumOld` | `String` | 否 | 上期持股数 |
| `reportDate` | `String` | 否 | 报告期 |
| `announcementDate` | `String` | 否 | 公告日期 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
# 调用接口
data = api.query_kjj_data(
    interfaceId="45",
    stockCode=None,  # String  # 默认: 
    dataDate=None,  # String  # 默认: 
    holderType=None,  # Integer  # 默认: 
)
```
