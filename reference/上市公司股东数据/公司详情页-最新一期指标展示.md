# 公司详情页-最新一期指标展示

> 分类: 上市公司股东数据 | 目录: `上市公司股东数据` | 索引见 `SKILL.md`


### 描述

公司详情页-最新一期指标展示

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "LISTEDCOMPANY_LATEST_FINANCE_INDEX_4A98062C" | 接口ID |
| `stockCode` | `String` | 是 | - | 股票代码 示例：000001 |

### 返回参数

ResultData.data 为 ListedCompanyFinanceVo

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `PE` | `BigDecimal` | 否 | 市盈率（TTM） |
| `PB` | `BigDecimal` | 否 | 市净率 |
| `eps` | `BigDecimal` | 否 | 每股收益 |
| `bps` | `BigDecimal` | 否 | 每股净资产 |
| `totalRevenue` | `BigDecimal` | 否 | 总营收 |
| `revenueYoy` | `BigDecimal` | 否 | 营收同比 |
| `netProfit` | `BigDecimal` | 否 | 归母净利润 |
| `netProfitYoy` | `BigDecimal` | 否 | 归母净利润同比 |
| `netProfitDeduct` | `BigDecimal` | 否 | 扣非净利润 |
| `netProfitDeductYoy` | `BigDecimal` | 否 | 扣非净利润同比 |
| `grossMargin` | `BigDecimal` | 否 | 毛利率 |
| `netMargin` | `BigDecimal` | 否 | 净利率 |
| `roe` | `BigDecimal` | 否 | 净资产收益率 |
| `debtRatio` | `BigDecimal` | 否 | 资产负债率 |
| `dividedYield` | `BigDecimal` | 否 | 股息率 |
| `dividedPayOutRatio` | `BigDecimal` | 否 | 股利支付率 |
| `reportDate` | `String` | 否 | 报告期 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
# 调用接口
data = api.query_kjj_data(
    interfaceId="LISTEDCOMPANY_LATEST_FINANCE_INDEX_4A98062C",
    stockCode=None,  # String  # 默认: 
)
```
