# 公司详情页-股东股本 -机构持仓明细

> 分类: 上市公司股东数据 | 目录: `上市公司股东数据` | 索引见 `SKILL.md`


### 描述

公司详情页-股东股本 -机构持仓明细

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "LISTEDCOMPANY_INSTITUTION_HOLDING_DETAIL_DBC4812E" | 接口ID |
| `stockCode` | `String` | 是 | - | 股票代码 示例：000001 |
| `reportDate` | `String` | 是 | - | 报告期 如 2025-09-30 |

### 返回参数

ResultData.data 为 List，元素为 InstitutionHoldingSummaryVo

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `reportDate` | `String` | 否 | 报告期 |
| `totalRatioOfCirculate` | `BigDecimal` | 否 | 占流通股比（绘制图表） |
| `price` | `BigDecimal` | 否 | 股价（绘制图表） |
| `ratioOfCycleChg` | `BigDecimal` | 否 | 持股环比变化 |
| `totalQuantity` | `Long` | 否 | 累计持股数量 |
| `totalMarketValue` | `BigDecimal` | 否 | 累计持股市值 |
| `totalInstitutionNum` | `Integer` | 否 | 累计机构数 |
| `totalListedShares` | `Long` | 否 | 总流通股本 |
| `type` | `Integer` | 否 | 类型： 0-全部 1-其他 2-基金 3-国家队 |
| `detailVoList` | `List<InstitutionHoldingDetailVo>` | 否 | 机构持仓明细数据 |

**`detailVoList`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `reportDate` | `String` | 否 | 报告期 |
| `ratioOfCirculate` | `BigDecimal` | 否 | 占流通股比（绘制图表） |
| `totalQuantity` | `Long` | 否 | 累计持股数量 |
| `holdName` | `String` | 否 | 股东名称 |
| `holdCode` | `String` | 否 | 基金代码 |
| `infoId` | `String` | 否 | infoId |

```python
from equal_data import KjjApi

    interfaceId="LISTEDCOMPANY_INSTITUTION_HOLDING_DETAIL_DBC4812E",
    stockCode=None,  # String  # 默认: 
    reportDate=None,  # String  # 默认: 
)
```
