# 主要财务指标

> 分类: 财务指标相关数据 | 目录: `财务指标相关数据` | 索引见 `SKILL.md`


### 描述

主要财务指标

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "1" | 接口ID |
| `stockCodeList` | `List<String>` | 是 | - | 股票代码列表 |
| `reportDate` | `String` | 否 | - | 报告期 如 2025-09-30，默认为最新一期 |

### 返回参数

ResultData.data 为 List，元素为主要财务指标

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `dateStr` | `String` | 否 | 报告期 |
| `publicDate` | `String` | 否 | 公告发布时间 |
| `eps` | `BigDecimal` | 否 | 基本每股收益（元） |
| `epsYoy` | `BigDecimal` | 否 | 每股收益同比 |
| `epsDeduct` | `BigDecimal` | 否 | 扣非每股收益（元） |
| `epsDiluted` | `BigDecimal` | 否 | 稀释每股收益（元） |
| `bps` | `BigDecimal` | 否 | 每股净资产（元） |
| `bpsTb` | `BigDecimal` | 否 | 每股净资产同比 |
| `epsCapital` | `BigDecimal` | 否 | 每股资本公积（元） |
| `epsUnallocatedProfit` | `BigDecimal` | 否 | 每股未分配利润（元） |
| `epsOperatingCashFlow` | `BigDecimal` | 否 | 每股经营现金流（元） |
| `returnOnEquity` | `BigDecimal` | 否 | 净资产收益率（加权） |
| `returnOnEquityDeduct` | `BigDecimal` | 否 | 净资产收益率（扣非/加权） |
| `returnOnAssets` | `BigDecimal` | 否 | 总资产收益率（加权） |
| `returnOnCapital` | `BigDecimal` | 否 | 投入资本回报率 |
| `grossProfitMargin` | `BigDecimal` | 否 | 毛利率 |
| `netProfitMargin` | `BigDecimal` | 否 | 净利率 |
| `netCashFlowRatio` | `BigDecimal` | 否 | 销售净现金流/营业收入 |
| `operatingCashFlowRatio` | `BigDecimal` | 否 | 经营净现金流/营业收入 |
| `actualTaxRate` | `BigDecimal` | 否 | 实际税率 |
| `currentRatio` | `BigDecimal` | 否 | 流动比率 |
| `quickRatio` | `BigDecimal` | 否 | 速动比率 |
| `cashFlowRatio` | `BigDecimal` | 否 | 现金流量比率 |
| `debtRatio` | `BigDecimal` | 否 | 资产负债率 |
| `debtRatioYoy` | `BigDecimal` | 否 | 资产负债率同比 |
| `equityMultiplier` | `BigDecimal` | 否 | 权益乘数 |
| `equityDebtRatio` | `BigDecimal` | 否 | 产权比率 |
| `totalAssetsTurnoverDays` | `BigDecimal` | 否 | 总资产周转天数 |
| `inventoryTurnoverDays` | `BigDecimal` | 否 | 存货周转天数 |
| `receivablesTurnoverDays` | `BigDecimal` | 否 | 应收账款周转天数 |
| `totalAssetsTurnover` | `BigDecimal` | 否 | 总资产周转率 |
| `inventoryTurnover` | `BigDecimal` | 否 | 存货周转率 |
| `receivablesTurnover` | `BigDecimal` | 否 | 应收账款周转率 |
