# 主营业务数据分析历史数据

> 分类: 财务指标相关数据 | 目录: `财务指标相关数据` | 索引见 `SKILL.md`


### 描述

主营业务数据分析历史数据

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "2" | 接口ID |
| `stockCodeList` | `List<String>` | 是 | - | 股票代码列表 |
| `reportDate` | `String` | 否 | - | 报告期,如 2025-12-31，默认为最新一期 |
| `filterType` | `Integer` | 否 | `1` | 筛选方式 1-按年报 2-按中报 |

### 返回参数

ResultData.data 为 List，元素为主营业务分析历史数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `dateStr` | `String` | 否 | 报告期 |
| `list` | `List<EastStockBusinessAnalysisVo>` | 否 | 主营业务分析列表 |

**`list`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `reportTime` | `String` | 否 | 报告期 |
| `classType` | `String` | 否 | 分类类型 |
| `rank` | `Integer` | 否 | 排名 |
| `value` | `String` | 否 | 值 |
| `change` | `String` | 否 | 变化 |
| `changeRate` | `String` | 否 | 变化率 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
token = os.getenv('KJJ_API_KEY')  
# 调用接口
data = api.query_kjj_data(
    interfaceId="2",
    stockCodeList=None,  # List<String>  # 默认: 
    reportDate=None,  # String  # 默认: 
    filterType=None,  # Integer  # 默认: 1
)
```
